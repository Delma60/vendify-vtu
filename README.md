# VTU Platform

This workspace holds the VTU platform project scaffold for the `vtu-web` and `vtu-mobile` applications.

Phase 0 focuses on project initialization, baseline architecture, and environment configuration.

## Design plan
- Palette: white background, vibrant orange-green gradient hero, warm coral/orange primary (`#F97316`), fresh green accent (`#22C55E`), dark charcoal text (`#111827`)
- Type: large, bold, friendly headlines. No jargon.
- Signature: floating "quick top-up" card in the hero that feels like you can use it right now.
- Language: everyday Nigerian English. Use phrases like "Top up in seconds", "Pay your bills", "Send money".

## Projects

- `vtu-web` — Next.js 14 App Router web + API project
- `vtu-mobile` — Expo Router React Native mobile app

## Next steps

1. Install dependencies in each project
2. Run `npm run dev` from `vtu-web`
3. Run `npm start` from `vtu-mobile`
